# 一木记物 - 版本记录

## v1.0.0 (2026-06-10)
- 初版发布
- 物品 CRUD：添加/编辑/删除物品
- 分类管理：一级/二级分类，默认 8 类 + 15 子类
- 物品模板：二手资产/虚拟资产/临期商品/金银首饰
- 展示模式：列表/瀑布流/极简，首页右上角切换
- 排序：默认/价格/购买日期/到期日/日均成本，支持升降序
- 统计页：总览卡片、分类饼图、日均成本 TOP、月度柱图
- 日历视图：只显示购买日期标记
- 物品详情：基本信息、成本分析、时间线、库存/借出开关
- 购买渠道(淘宝/京东/闲鱼等10个)+购买方式(全款/分期/合租)
- 保修期：天/月/年/永久 + 过保日期
- 额外费用 + 费用说明，计入总价和日均成本
- 封面图：相册上传 + 百度图片搜索
- WebDAV 备份/恢复/检测连接
- 双主题支持：Material Design + Miuix
- 标签筛选：一级分类 → 二级子分类联动
- 日期中文化：日期选择器/过保时间全中文
- Release APK: 54.1MB
